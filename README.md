## 实现功能
底部路由组件，首页轮播图，商家或食品搜索，用户订单，用户评价，个人详情，退出登录，购物车，点餐，结算，添加用户收货地址，选择用户收货地址，注册，登录等

## 使用
1. npm install下载依赖
2. npm run dev启动前台（需要先启动后台delivery_node）
